🔍 Saisie des ASINs Amazon

Renseigner l'ASIN pour chaque produit — utilisé pour récupérer les images depuis Amazon.fr

Comment trouver un ASIN ?
1. Cliquer 🔍 pour ouvrir la recherche Amazon.fr dans un nouvel onglet
2. Choisir le bon produit dans les résultats
3. Copier l'ASIN depuis l'URL : amazon.fr/dp/B0XXXXXXXXX/...
4. Coller dans le champ et appuyer sur Tab pour passer au suivant
5. Cliquer Générer asins.json → copier dans scripts/asins.json → lancer node scripts/set-asins.js
