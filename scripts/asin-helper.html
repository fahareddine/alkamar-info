🔍 Saisie des ASINs Amazon

Renseigner l'ASIN pour chaque produit afin de récupérer 3 photos distinctes depuis Amazon.fr

Comment trouver l'ASIN ?
1. Cliquer 🔍 pour ouvrir la recherche Amazon.fr
2. Choisir le bon produit dans les résultats
3. Copier l'ASIN depuis l'URL : amazon.fr/dp/B0XXXXXXXXX/...
4. Coller dans le champ ci-dessous
5. Cliquer Générer asins.json puis copier dans scripts/asins.json
